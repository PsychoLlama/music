Song.js

A JavaScript library for managing playlists

Creating a playlist...

To begin, we need to create a playist object. Here's how:

var playlistMgr = new Song(playlist);

We'll format your playlist in a moment. for now, let's have an overview. Song takes your playlist data and creates <audio> tags, pushing them into a playlist property.

playlistMgr.playlist
> [ <audio> <audio> ... ]

Formatting the playlist

The Song constructor takes in a list of songs as an object array. Each object represents a song. 

[{
  title: "title string",
  src: "URL string",
  img: "URL string"
}, {
  title: "Second song",
  src: "URL",
  img: "URL"
}]

A src property is required in each song object. All other properties are optional.
